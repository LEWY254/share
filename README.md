# Share - App Testing Platform

A React + Supabase app for sharing beta apps with testers and collecting rich feedback.

## Features

- Role-based access (admin, developer, tester)
- APK and app uploads with screenshots
- Feedback collection with ratings
- OAuth authentication (Google, GitHub)

## Tech Stack

- React + Vite + TypeScript
- Tailwind CSS
- Supabase

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and configure your Supabase credentials.
